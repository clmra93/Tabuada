# Tabuada

## Sobre o projeto.
Este foi um projeto criado pensando em como ajudar a crianças e qualquer pessoa que queira se aperfeiçoar em tabuada matemática.
A ideia surgiu vendo minha filha estudando tabuada para atividades da escola. Sabendo que crianças hoje tem mais facilidade em aprender quando algo está ligado a tecnologia, pensei em unir o útil ao agradável. Já que ela gosta de usar o computador, fiz esse projeto para que possa tornar o tempo em frente a tela produtivo.

## Informações:
Ao fazer o git clone do repositório, você encontra o aplicativo na pasta dist.
O uso é livre, utilize, faça as alterações que julgar necessárias, divirta-se, e espero que seja útil para vocês.
